# Ikony aplikacji PWA

Dla poprawnego działania aplikacji PWA wymagane są ikony w formatach PNG.

Proszę umieścić w tym folderze dwa pliki:

1.  `icon-192.png` - ikona o wymiarach 192x192 pikseli.
2.  `icon-512.png` - ikona o wymiarach 512x512 pikseli.

Możesz wygenerować ikony korzystając z narzędzi online, np. [PWA Icon Generator](https://tools.crawlink.com/tools/pwa-icon-generator/).

Obecnie pliki `icon-192.png` i `icon-512.png` są pustymi plikami zastępczymi.
